## 📂 Curriculum & Lab Milestones

---

### 1) [Lab 01-02] 🧮 Utility Logic & UI Foundations

- Pro Calculator: Developed a manual arithmetic engine avoiding eval() for secure calculations.
- Smart Timer: Implementation of real-time temporal tracking with robust lifecycle cleanup.
- Concepts: Flexbox Grid layouts, state-driven UI updates, and resource management.

### 2) [Lab 03] 🌌 Synced-Core Dynamic Theme Hub

- Global Theme Engine: A centralized Context-Reducer pattern that synchronizes Dark/Light modes across the entire navigation stack.
- Asynchronous Pipeline: Integration with JSONPlaceholder API using standardized Axios calls.
- UX Engineering: Parallax scroll effects and high-performance image caching.

### 3) [Lab 04] 🛡️ Advanced Middleware & Background Engine
- Network Interceptors: Centralized apiClient logic to monitor, log, and handle global API errors (404/500) automatically.
- Native Task Manager: Integration with OS-level task schedulers for background data synchronization.
- Simulation Engine: Engineered a custom interval-based logic to verify background tasks within the Expo Go environment.
